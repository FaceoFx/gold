# Gold Market Live Tracker

A modern live gold price tracking platform focused on the Eastern Province market in Saudi Arabia.
The platform provides real-time gold and silver pricing, automatic updates, regional currency conversion, and references from trusted gold sources. ([faceofx.github.io][1])

## 🌍 Live Demo

[Gold Market Live Demo](https://faceofx.github.io/gold/?utm_source=chatgpt.com)

---

# ✨ Features

* 🔴 Live gold price updates every 60 seconds
* 🥇 Gold prices by karat:

  * 24K
  * 22K
  * 21K
  * 18K
* 🥈 Live silver pricing
* 💱 Multi-currency support:

  * SAR
  * AED
  * KWD
  * QAR
  * BHD
  * OMR
  * USD
  * EUR
  * GBP
  * and more
* 📈 Real-time ounce price tracking
* 🧮 Automatic gram price calculation
* 🌐 External trusted references integration
* 📱 Responsive mobile-friendly UI
* ⚡ Auto-refresh live broadcasting mode

# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/FaceoFx/gold.git
```

Open the project folder:

```bash
cd gold
```

Run locally using any static server.

Example:

```bash
python -m http.server
```

Then open:

```bash
http://localhost:8000
```
