## ✨ **Features**

### 📊 **Real-Time Gold Prices**
- Live gold price tracking with 60-second auto-refresh
- Support for **14 currencies** (SAR, AED, KWD, QAR, BHD, OMR, JOD, EGP, USD, EUR, GBP, INR, PKR, TRY)
- **4 karat types** (24K, 22K, 21K, 18K)
- Silver price tracking

### 🤖 **AI-Powered 24-Hour Prediction**
- Advanced prediction system using real market data
- Technical analysis (Moving Averages, RSI, Momentum, Support/Resistance)
- USD Index integration for accurate forecasting
- Shows expected price movement in **SAR amounts** (not percentages)
- Confidence scoring (60-95%)
- **Never shows 50/50** - always provides directional bias

### 📈 **Interactive Price Chart**
- Beautiful 48-hour price history chart
- Real-time updates with smooth animations
- Color-coded trends (green for up, red for down)
- Responsive design for all screen sizes

### 🔥 **Firebase Cloud Storage**
- Unlimited price history (up to 30 days)
- Data persists forever (never lost)
- Cross-device synchronization
- Automatic hourly backups

### 💰 **Smart Calculators**
- **Gram/Karat Calculator**: Calculate prices for any weight and karat
- **Buy/Sell Calculator**: Includes manufacturing fees (تلبيس)
- **Custom Weight Calculator**: Flexible weight input (grams, tola, ounce)

### 📱 **Progressive Web App (PWA)**
- Works offline
- Install on mobile/desktop
- Fast loading with service worker caching
- Native app-like experience

### 🌍 **Arabic-First Design**
- Full RTL (Right-to-Left) support
- Beautiful Arabic typography
- Optimized for Saudi Arabia and GCC region
- Dark theme with gold accents

### ⚡ **Performance**
- Lightning-fast loading
- Optimized for mobile networks
- Minimal bandwidth usage
- Smart API fallback system (3 backup APIs)

## 🛠️ **Technology Stack**

- **Frontend**: Pure HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Firebase Realtime Database
- **APIs**: 
  - Gold-API.com
  - CoinGecko
  - Yahoo Finance
  - ExchangeRate-API
- **Charts**: HTML5 Canvas
- **Fonts**: IBM Plex Sans Arabic, IBM Plex Mono
- **PWA**: Service Worker, Web Manifest

---

## 🚀 **How It Works**

1. **Fetches real-time gold prices** from multiple APIs every 60 seconds
2. **Analyzes market data** using technical indicators and USD strength
3. **Generates AI predictions** for next 24 hours with confidence scores
4. **Saves data to Firebase** every hour for unlimited history
5. **Displays beautiful charts** with 48-hour price movements
6. **Updates automatically** with countdown timer

---

## 📊 **Prediction System**

The AI prediction system uses:
- **22%** Price trend analysis
- **18%** RSI (Relative Strength Index)
- **18%** Momentum indicators
- **12%** Support/Resistance levels
- **30%** USD Index (real market data)

**Result**: Accurate directional forecasts with 60-95% confidence
