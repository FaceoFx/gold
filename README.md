## نبذة سريعة

تطبيق ويب عربي لمتابعة أسعار الذهب لحظيًا في المنطقة الشرقية، مع دعم العملات الخليجية والعالمية، مخطط سعري مباشر، حاسبة ذهب، ومؤشر توقع مبسط لحركة السعر خلال 24 ساعة.

> التوقعات المعروضة تحليلية فقط وليست توصية شراء أو بيع.

---

## أهم المميزات

- تحديث تلقائي للأسعار كل **30 ثانية**.
- عرض سعر الأونصة وجرام الذهب حسب العيارات: **24، 22، 21، 18**.
- دعم عدة عملات، منها: **SAR, AED, KWD, QAR, BHD, OMR, USD, EUR, GBP**.
- مخطط حركة السعر خلال آخر **48 ساعة**.
- مؤشر حالة البورصة: مفتوحة / مغلقة، مع العدّ التنازلي للإغلاق أو الفتح.
- توقع 24 ساعة مبني على بيانات السوق ومؤشرات فنية، مع تثبيت وسلاسة لتقليل تغيّر الأرقام العشوائي عند تحديث الصفحة.
- حاسبة ذهب لحساب السعر حسب العيار، الوزن، والمصنعية.
- تصميم عربي RTL متجاوب ومناسب للموبايل.
- دعم PWA للتثبيت على الهاتف والعمل كتطبيق.

---

## التقنيات المستخدمة

- **HTML5 / CSS3 / JavaScript**
- **Firebase Realtime Database** لحفظ سجل الأسعار
- **Canvas Charts** لرسم المخططات
- **Service Worker + Manifest** لدعم PWA والكاش
- خطوط: **IBM Plex Sans Arabic** و **IBM Plex Mono**

## مصادر البيانات

يعتمد التطبيق على أكثر من مصدر واحتياطي لتحسين الاستقرار، منها:

## طريقة التشغيل محليًا

افتح ملف `index.html` مباشرة في المتصفح، أو شغّل المشروع عبر سيرفر محلي بسيط:

```bash
python -m http.server 8000
```

ثم افتح:

```text
http://localhost:8000
```
